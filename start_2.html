Політика конфіденційності
Що таке CVV / CVC?

Це останні три цифри, вказані на зворотній стороні картки.

Для карт Amex вони знаходяться на передній панелі.

Навіщо це вказувати?

CVV-код потрібен для перевірки справжності Вашої картки під час оплати.

Також він убезпечить вас при проведенні онлайн платежів.

Верифікація карти

Верифікація банківської карти необхідна для підтвердження володіння карти користувачем, а також для того, щоб переконатися в статусі активності карти.

З рахунку Вашої картки буде проведена транзакція на випадкову суму, яка після успішної верифікації буде повернута на рахунок.

Для можливості проведення перевірки, вкажіть доступний залишок по картці.
